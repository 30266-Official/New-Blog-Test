# New-Blog-Test

这是一个用于学习和测试 Nuxt.js 的个人博客项目样例（教学/实验用途）。项目基于 Nuxt 4 + Vue 3，旨在演示页面、布局和响应式主题的实现。

**快速开始**
- **安装依赖**: 在项目根目录运行:

```powershell
npm install
```

- **本地开发**: 启动开发服务器:

```powershell
npm run dev
```

- **构建**:

```powershell
npm run build
npm run preview
```

**项目结构（简要）**
- `app/`：Nuxt 应用源码（页面、布局、组件、assets 等）
- `pages/`：页面文件（路由由文件系统自动生成）
- `layouts/`：页面布局（包含 `bottom_bar.vue` 等）
- `public/`：静态资源（如 `robots.txt`）
- `nuxt.config.ts`：Nuxt 配置文件

**已知信息**
- 使用的 Nuxt 版本：在 `package.json` 中为 `^4.2.1`
